# Integrating Fabricate and Ephemeral using MySQL

## Prerequisites

- Node.js v20 or later
- A Fabricate API key
- An Ephemeral API key

## Setup

Install dependencies

```bash
yarn install
```

Copy .env.example to .env and set your API keys and database parameters:

```bash
cp .env.example .env
```

## Run

Run the script:

```bash
yarn start
```
